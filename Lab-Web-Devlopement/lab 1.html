5. Search Button

Search on Google

In this code: We have a basic HTML structure with a form that contains an input field and a search button. The form element has an action attribute set to "https://www.google.com/search", which specifies the URL to Google's search page. The method attribute is set to "GET" to send the search query as a URL parameter. The target attribute is set to "_blank", which opens the Google search page in a new browser tab or window when the search button is clicked. Inside the form, there is an input element of type "text" with the name attribute set to "q," which is the parameter used by Google for the search query. Finally, there is an input element of type "submit" with the value "Search" that serves as the search button. When you enter a search term and click the "Search" button, it will take you to Google's search results page with the search query you entered.
